# Shopping PC GK

Frontend React cho web ban hang nho, su dung API tu backend Node + MongoDB.

## Chuan bi

1. Dam bao backend dang chay o `http://localhost:5000`
2. Tao file `.env` trong thu muc nay:

```bash
REACT_APP_API_URL=http://localhost:5000/api
```

## Chay du an

```bash
npm install
npm start
```

Mo `http://localhost:3000` de su dung.

## Tinh nang

- Hien thi danh sach san pham
- Them/xoa/tang giam so luong trong gio hang
- Dat hang voi thong tin nhan hang
- Hien thi thong bao loi/thanh cong sau khi dat hang
